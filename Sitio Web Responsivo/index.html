Asignatura: Programación Web

Propósito de aprendizaje

El alumno implementará sistemas empleando tecnologías web para extender la disponibilidad y alcances de los servicios de la organización.

Cuatrimestre: Séptimo

Unidades de Aprendizaje

  1. Introducción a los sistemas web
  2. Programación del lado del cliente
  3. Programación del lado del servidor
  4. Autenticación de usuarios

Temas por Unidad de Aprendizaje

Unidad I. Introducción a los sistemas web

Modelo cliente-servidor, HTTP/HTTPS, URL, TCP/IP, FTP, tecnologías del lado del cliente y servidor, configuración de servidores web.

Unidad II. Programación del lado del cliente

HTML5, etiquetas semánticas, CSS3, diseño responsivo, JavaScript y manipulación del DOM.

Unidad III. Programación del lado del servidor

Lenguajes del lado del servidor (PHP, Python, .NET, JEE), APIs, servicios web, programación en tres capas y monitoreo de aplicaciones.

Unidad IV. Autenticación de usuarios

Implementación de mecanismos de seguridad y manejo de excepciones en servidores web.

Competencia profesional

Desarrollar soluciones innovadoras de integración de tecnologías de la información mediante metodologías de desarrollo de software, diseño de bases de datos y seguridad de la información.
